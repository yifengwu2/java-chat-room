# 🌐 Java 聊天室系统（基于 Socket）

一个使用 **纯 Java SE** 实现的多人在线聊天室，支持多用户并发通信、私聊、在线查询等功能。适合 Java 网络编程初学者学习与参考。

> 💡 无需框架，仅用 `Socket` + `多线程` + `IO流` 构建完整的 C/S 架构应用。

---

## 🚀 功能特性

- ✅ **TCP 通信**：基于 `ServerSocket` 和 `Socket` 实现可靠连接
- ✅ **多客户端并发**：每个客户端独立线程处理，支持多人同时在线
- ✅ **群聊广播**：消息自动发送给所有在线用户
- ✅ **私聊功能**：支持 `@用户名 消息` 格式进行点对点聊天
- ✅ **自定义昵称**：用户可输入昵称登录（支持中文）
- ✅ **上下线通知**：用户加入/离开时自动广播通知
- ✅ **命令系统**：
  - `/users`：查看当前在线用户列表
  - `/whois <昵称>`：查询某用户是否在线
  - `/help`：显示帮助信息
  - `/quit` 或 `bye`：退出聊天
- ✅ **时间戳**：每条消息显示 `[HH:mm:ss]` 时间
- ✅ **线程安全**：使用 `synchronized` 保证共享资源安全
- ✅ **UTF-8 编码**：支持中文消息传输

---
